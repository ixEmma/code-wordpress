# 1st Providence Style Guide

## Visual Direction

The rebuild uses a clean healthcare education style: clear hierarchy, strong Providence blue sections, dark red calls to action, practical imagery, and restrained spacing.

## Colors

- Providence blue: `#004990`
- Dark red: `#A72D34`
- Button hover red: `#8F252B`
- Dark navy: `#001932`
- Light gray: `#F1F1F1`
- Off white: `#F7F9FB`
- White: `#FFFFFF`
- Main text: `#1F2933`

## Typography

Use Nunito Sans through Elementor Site Settings.

- Hero H1: 45px desktop, 38px tablet, 32px mobile, 900 weight.
- Section H2: 34px desktop, 30px tablet, 26px mobile, 800-900 weight.
- H3/card heading: 22px desktop, 20px tablet, 19px mobile, 800 weight.
- Body: 16px desktop/tablet, 15px mobile, 1.6-1.7 line height.
- Buttons: 14px desktop/tablet, 13px mobile, uppercase, 800 weight.

## Buttons

Primary buttons:

- Background: `#A72D34`
- Hover: `#8F252B`
- Text: white
- Radius: 4-6px
- Letter spacing: 1.5px
- Desktop padding: 14px 30px
- Mobile padding: 12px 24px

## Layout

- Use Elementor Containers.
- Use max width around 1120px for most content sections.
- Use 980px for centered text sections.
- Alternate white and light gray sections for program pages where the reference does so.
- Use blue bands for major explanatory or benefit sections.

## Imagery

Use approved healthcare training, classroom, clinical, or program-specific imagery. Do not commit images to the public repository.

## Accessibility Notes

- Keep heading order logical.
- Maintain sufficient contrast on blue and dark image-overlay sections.
- Use descriptive image alt text during implementation.
- Buttons should use clear action labels.
