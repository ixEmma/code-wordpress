# Codex Instruction Authoring Skill

## Purpose

Use this skill when creating, rewriting, auditing, or improving instruction files for Codex.

This project uses Markdown instruction files to guide Codex when working on WordPress, Elementor, design systems, homepage rebuilding, JSON templates, debugging, and quality checks.

The goal is to create clear, reusable, task-specific instruction files that Codex can follow consistently.

## Core Rule

Do not write vague instruction files.

Every Codex instruction file should explain:

1. What the instruction file is for.
2. When Codex should use it.
3. What Codex must do first.
4. What steps Codex must follow.
5. What Codex must not do.
6. What output format Codex should use.
7. What quality checks must pass.
